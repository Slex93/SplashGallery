### Description

The app client with Image library using Unsplash api In order with dark theme support

### Tools

- MVVM
- Dagger 2
- Retrofit 2
- Coroutines
- Jetpack's pagination 3
- Navigation component (Jetpack's navigation)
- ViewPager2
- Glide
